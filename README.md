# Simple Rails-React CRUD Web Application

## Stack:

- Ruby 2.3.1
- Rails 5.0.6
- ReactJS 
- Sqlite3
 
## How to use:

- Clone the project
- Install RVM 
- Install Ruby on Rails
- Install gems and migrate the database
```
bundle install
rake db:migrate
```
- Run the app
```
rails s
```
- Open the app in your browser
```
http://localhost:3000/cars
```
